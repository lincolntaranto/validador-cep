## validador-cep

# Descricação

Verificador de CEP via CLI. Com ele você pode verificar seu CEP usando a API ViaCEP.

# Funcionalidades

- Verificar CEP
- Exibir informações do CEP

# Como executar

- git clone ...
- python -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt
- python main.py

# Como executar os testes

pytest
